# Description

The following is an implementation of Euler coordinates and Lagrangian coordinates for the deformation of a flexible plate for MRAG-I2D. Special thanks to my mentor David Gutiérrez and supervisor Wim van Rees for their indispensable guidance during my time as a UROP.


## How To Run!

The main entrypoint for MRAG-I2D simulation adjoint optimization is "main.cpp" and the latter plotting for image creation is in "flappingPlateOptimization.py". Please refer to the libraries/packages listed below for any software requirements for running the code. Please consult the bibliographic references to learn more about adjoint_optimization and MRAG-I2D!

## Bibliographic References

[Adjoint Optimization (See section 5.2)](https://www.researchgate.net/publication/305342885_Using_adjoint-based_optimization_to_study_kinematics_and_deformation_of_flapping_wings),
